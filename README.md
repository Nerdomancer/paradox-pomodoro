# Paradox Pomodoro

It's a simple, customizable pomodoro timer. Made with python and no external modules.

### Usage

`python3 paradoxpomo.py` for standard pomodoro timings (25 minutes of work, 5 minutes rest, 30 minutes rest every 4 pomodori)

```python3 paradoxpomo.py <work minutes> <rest minutes> <long rest minutes> <pomodori per cycle>``` for custom pomodoro configurations.
